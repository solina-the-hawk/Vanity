Vanity: Modular Character Description Manager (v1.1.0)

A centralized, zero-dependency script interface for Mudlet to easily swap and build character descriptions in Achaea.
Vanity cleanly organizes your saved descriptions, modular elements (hair, eyes, build), and poses into an easy-to-use in-client dashboard.
Features include temporary poses that automatically clear via GMCP when you move rooms, dynamic add-on strings for temporary gear or roleplay states, and a smart style-checker to prevent common description faux pas.
To adjust your color theme, character limits, or to toggle the game-echo gagging, simply open the Vanity script in your Script Editor and adjust the configuration block at the top.
You can also type "vanity help" in the game to see a list of on-the-fly commands and dashboard tools!
